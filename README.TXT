实验器材:
	MiniSTM32F103开发板V3版本
	
实验目的:
	使用STM32开发驱动单元测试台架，功能包括电机PWM调速，测速，六维力/力矩传感器数据采集，LCD屏实时显示。
	
硬件资源:
	1,PA0――电机测速
	2,PA8――PWM输出，LED显示
	3,ALIENTEK 2.8/3.5/4.3/7寸TFTLCD模块(通过GPIO驱动,连接关系见lcd.h) 
	4,ADC(STM32内部ADC，PA123,PC023)
	5,UART串口数据传输（PA9,PA10）
	
存在的问题：
	1.参考资料（https://blog.csdn.net/weixin_41939983/article/details/105861840 ），仅作PC0/1/2/3通道的AD转换，但部分通道存在问题，可能是LCD引起的问题；
	2.原本的32板无故烧毁，目前没有可测试的硬件；（已适用新购买的开发板进行测试）
	3.电机转速测量部分可能存在问题；
	4.可尝试添加按键功能。
	5.发现是否适用LCD对ADC转换数值会有很大影响，因此需要考虑在试验台上进行验证。
